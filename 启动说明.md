# 项目启动说明

## 项目信息
- **项目名称**: zhengfucaigou (政府采购招标网站)
- **技术栈**: Next.js 16.2.10 + React 19 + TypeScript + Tailwind CSS
- **端口**: 4003

## 启动步骤

### 方式一：使用 npm (推荐)

1. 打开命令行工具（CMD 或 PowerShell）
2. 进入项目目录：
   ```bash
   cd D:\kehu\zhaobiao
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 访问应用：
   - 本地访问：http://localhost:4003
   - 网络访问：http://[你的IP]:4003

### 方式二：使用启动脚本

**Windows 用户：**
双击 `start.bat` 文件（如果存在）

**Linux/Mac 用户：**
```bash
./start.sh
```

## 常见问题

### 端口被占用
如果 4003 端口已被占用，可以修改 `package.json` 中的端口号：
```json
"scripts": {
  "dev": "next dev -p [新端口]",
  "start": "next start -p [新端口]"
}
```

### 依赖安装失败
尝试清理并重新安装：
```bash
rm -rf node_modules package-lock.json
npm install
```
